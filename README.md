# E-commerce Api.

Api desarrollada para e-commerce con diversas funcionalidades como: 
- Inicio de sesión
- Inserción de productos nuevos (Crud)
- Visualización de productos
- Manejo de roles (Usuario / Administrador) 

### Tecnologías utilizadas:

- Node. js
- MongoDb
- JWT
- Express
- Multer
